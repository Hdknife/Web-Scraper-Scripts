# Assignment

## Overview

This assignment consists of a Python project with three main files: `main.py`, `constants.py`, and `google_search.py`. The purpose of this project is to collect data from the EventBrite 
website and store it in a CSV file named `events.csv`. The `main.py` file contains the `EventBrite` class, which is responsible for web scraping using BeautifulSoup and collecting event data.

## Requirements

This project requires the following libraries/modules:

- `beautifulsoup4` (bs4)
- `requests`
- `csv`
- `time`
- `random`

You can install these dependencies using the following command:

```bash
pip install beautifulsoup4 requests
